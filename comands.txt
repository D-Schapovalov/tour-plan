npm i установить все пакеты (gulp)

npm run dev запусить в режиме разработчика

npm run build запусить в режиме продакшн (для заказчика)

npm run zip сделать архив проекта


порядок действий
В папку fonts закидываем необходимые шрифты в формате ttf и/или otf.
В папку с картинками кидаем все необходимые изображения (в процессе работы можно докидывать, если что-то пропустил или забыл).

1.npm i -установка

2.npm run dev -верстаем сайт, делаем все что нужно. Когда проект готов, останавливаем работу gulp комбинацией клавиш ctrl+C и запускаем команду №3.

3.npm run build -в этом режиме создаются все необходимые файлы в нужных форматах и разрешениях. Получается готовый проект который можно выгрузить на хостинг или отдать заказчику. Выполняем команду №4.

4.npm run zip -получаем  zip-архив с сайтом.
